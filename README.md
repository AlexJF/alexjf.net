## Setup

* Install system-level dependencies:
    * `python`:
        * Vanilla + virtualenv + lxml: Create virtualenv and use pip to load requirements.txt.
        * Anaconda: `conda env create -f alexjf.net.yml`.
    * `npm` 
* Run `.\make[.py/.ps1] setup`
